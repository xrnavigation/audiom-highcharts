# Audiom-Highcharts

A Highcharts plugin that adds [Audiom](https://audiom.net) audio accessibility to any Highcharts map.

This repository is a pnpm monorepo containing:

- [`plugin/`](plugin/) — the `audiom-highcharts` library (UMD + ESM build).
- [`sample/`](sample/) — a minimal Highcharts choropleth used as a development harness for the plugin.

## Getting Started

```bash
npm install
npm run dev:sample
```

See [the implementation issue](https://github.com/Coughlan-Lab/Audiom-Front-End/issues/1352) for full design context.
